STL: Algoritmul "upper_bound"

Utilizare pe vectori
Secventa urmatoare utilizeaza upper_bound pentru a determina pozitiile pe care se pot insera valorile 5 si 6 in sirul a, cu pastrarea relatiei de ordine.
    int a[] = { 2, 3, 5, 5, 5, 8, 10 };
    int * p = upper_bound (a, a+7, 5};
    cout << "pozitia " << p-a << " " << "valoarea actuala " << *p;
    p = upper_bound (a, a+7, 6);
    cout<< " pozitia " << p-a << " " << "valoarea actuala " << *p;
